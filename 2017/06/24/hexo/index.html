Hexo 搭建个人博客

环境准备

  • node.js 安装ok
  • [blogname].github.io 博客库创建ok

安装Hexo

npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server

打开本地 http://localhost:4000/ 查看效果

将 Hexo 博客推送到 Github 仓库上

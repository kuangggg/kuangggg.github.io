url 处理

http_build_query

生成 URL-encode 之后的请求字符串

parse_url

解析一个 URL 并返回一个关联数组，包含在 URL 中出现的各种组成部分。省略 第二个常量参数，返回数组键可能有以下几种

  • scheme (协议)
  • host
  • port
  • user
  • pass
  • path
  • query (? 之后的查询字串)
  • fragment (# 之后的锚点之类散列符号)

parse_str

将字符串解析成多个变量,如果设置了第二个变量 arr.变量将会以数组元素的形式存入到这个数组,作为替代.

文件路径相关

pathinfo

返回文件路径的信息

basename

返回路径中的文件名部分

dirname

返回路径中的目录部分

字符串相关

strtolower/strtoupper/ucfirst/ucwords

字符串全部字符转化为小写/大写/字符串首字母转化为大写/将字符串中每个单词的首字母转换为大写

strlen/mb_strlen

str_split/explode

将字符串按指定长度切割成数组/按照分隔符切分字符串成数组

strpos/stripos/strrpos

查找字符串首次出现的位置/最后一次出现位置/忽略大小写

substr

返回字符串的字串

substr_replace

替换字符串的字串

sprintf

返回格式化的字符串

  • %b 二进制
  • %c ASCII 字符
  • %f 浮点型
  • %s 字符串
  • %o 八进制
  • %x 十六进制
  • %d 十进制
  • %% 百分号

htmlentities

将所有可以转化 html 实体符号的都转化

htmlspecialchars

将特殊字符转换为 HTML 实体

  • & &amp;
  • " &quot;
  • ' &apos;
  • < &lt;
  • > &gt;

strip_tags

从字符串中去除 HTML 和 PHP 标记

trim/rtrim/ltrim

去除空白符

str_shuffle

随机打乱一个字符串

strrev

翻转字符串

数组相关

正则

preg_split

正则匹配分割字符串

call_user_func

同 call_user_func_array 差不多，只是除过第一个参数是回调函数，其余都是参数,
对于只有一个参数的回调或者没有参数的回调使用比较合适

func_get_args

返回函数参数列表的数组

func_num_args

获取函数传入参数数目

func_get_arg

获取参数列表某一项

杂项

extension_loaded

检查一个扩展是否已经加载

file

将文件读入到数组中
